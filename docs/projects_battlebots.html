Electric Longboard

As part of a summer workshop at MIT, my team and I built a pressure-sensitive electric longboard. If you lean forward there is more pressure on the front so the motors are driven forwards and when you lean back the motors stop using regenerative braking.

My involvement in the project was to write the Arduino code that took the values of the pressure sensors to actuate the motors as well as wiring the motor controllers to the hub-motors to then test them.
